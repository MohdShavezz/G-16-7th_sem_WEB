SET count 0
INCR count //1
INCRBY count 5  //6
INCRBYFLOAT count 1.223  //7.233333333333

#HASHES (objs)
HSET user:1 name shavez emial email@mgali.com age 22
HGET user:1 name
HMGET user:1 name email age
HINCRBY user:1 age -1 (inc/dec)
HGETALL user:1
HEXISTS user:1 name

#LIST (array)
LPUSH/RPUSH list element
LPOP/RPOP list
LTRIM list 0 2 (remove element, 0 1 2 remains)
LRANGE list 0 -1 (return all list data)
LLEN list (return size of list)

#SET
SADD set1 1 2 2 a  (1 2 a stored)
SREM set1 1 2
SMEMBERS set1 (returns all data)
SCARD set1 (set size)
SINTER set1 set2 (common ele get)

#SORTED SET
ZADD score 10 user10 5 user5
ZREM score user5
ZRANGE|ZREVRANGE score 0 -1
ZINCRBY score 2 user5
ZRANK|ZREVRANK score user5
ZREVRANGE score 0 -1 WITHSCORES
